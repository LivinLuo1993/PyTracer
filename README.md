  * Geometry:
    * Sphere intersection [✓]
    * Plane intersection [✓]
    * Cube Intersection
    * Triangle intersection
    * Quad intersection

  * Camera:
    * Camera exposure

  * Light:
    * Point light [✓]
    * Area light [Disk light working]
    * Radiance unit

  * Material:
    * Diffuse / Lambert [✓]
    * Mirror [✓]
    * Glossy
    * Glass

  * Core:
    * Camera Ray [✓]
    * Shadow Ray / Direct Lighting [✓]
    * Render Equation integration
    * Recursive tracing

  * Optimization:
    * multiprocessing [✓]
    * Anti-aliasing [Need improvement]

  * Imaging:
    * Gamma Correction [✓]
